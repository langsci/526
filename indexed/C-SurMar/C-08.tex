\scp{\tsc{South Halmahera-West New Guinea}}{C-08}
SHWNG languages are spoken in areas where multiple different
kinds of marsupials are found: cuscus (\it{Phalangeridae}),
bandicoots (\it{Peramelidae}),
as well as tree kangaroos (\it{Dendrolagus}) and wallabies (\it{Macropodiae}).
See \srf{14-01} for a more detailed overview.
