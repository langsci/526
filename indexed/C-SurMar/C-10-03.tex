\sscp{Other Papuan languages}{C-10-03}
The following languages are all non-Austronesian,
belonging to various Papuan language families.
See appendix \ref{ch:LanLis} for more detailed classification.
The data and languages in this section have not undergone as much
cross checking as that in previous sections as it was tangential
to our main focus.
Additionally, the number of languages combined with the number
of different marsupial species means it is not feasible to list
all the terms from these languages.
